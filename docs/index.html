Quick start

1
Install the driver
Install Visual C++ Redistributable 2015–2022, then the MIG driver matching your board (S2 / S6). See Driver Installation.
2
Install CameraMaster
Run the CameraMaster installer and launch the application. The Grabber connection status should turn green.
3
Load a module configuration
Pick a .ini file that matches your module / SER / DES / video format and review the contents in CameraMaster.
4
Bring up the camera
Power the module, initialize the sensor, and check live video. From here, run inspection items or integrate via the SDK.
